### Быстрая сортировка (Quick Sort)

###### Эффективный алгоритм сортировки, который выбирает "опорный" элемент и делит массив на части, меньшие и большие опорного элемента, и затем рекурсивно сортирует эти части.

```ts
function quickSort(arr: number[]): number[] {
  /* Если длина массива меньше 2, возвращаем массив */
  if (arr.length < 2) return arr;

  /* Выбираем случайный элемент в качестве опорного */
  const pivot = arr[Math.floor(Math.random() * arr.length)];

  /* Элементы массива, меньшие опорного */
  const less = arr.filter((x) => x < pivot);

  /* Элементы массива, большие опорного */
  const greater = arr.filter((x) => x > pivot);

  /* Рекурсивная сортировка меньших элементов, опорный элемент и рекурсивная сортировка больших элементов */
  return quickSort(less).concat(pivot).concat(quickSort(greater));
}

/* Тесткейсы для функции quickSort */

/* Тесткейс 1: Пустой массив */
console.log(quickSort([])); /* Ожидаемый результат: [] */

/* Тесткейс 2: Один элемент */
console.log(quickSort([1])); /* Ожидаемый результат: [1] */

/* Тесткейс 3: Два элемента */
console.log(quickSort([2, 1])); /* Ожидаемый результат: [1, 2] */

/* Тесткейс 4: Упорядоченный массив */
console.log(
  quickSort([1, 2, 3, 4, 5])
); /* Ожидаемый результат: [1, 2, 3, 4, 5] */

/* Тесткейс 5: Обратно упорядоченный массив */
console.log(
  quickSort([5, 4, 3, 2, 1])
); /* Ожидаемый результат: [1, 2, 3, 4, 5] */

/* Тесткейс 6: Массив с повторяющимися элементами */
console.log(
  quickSort([4, 1, 3, 2, 3])
); /* Ожидаемый результат: [1, 2, 3, 3, 4] */

/* Тесткейс 7: Массив с одинаковыми элементами */
console.log(
  quickSort([2, 2, 2, 2, 2])
); /* Ожидаемый результат: [2, 2, 2, 2, 2] */

/* Тесткейс 8: Большой массив случайных чисел */
const largeArray = Array.from({ length: 1000 }, () =>
  Math.floor(Math.random() * 1000)
);
console.log(
  quickSort(largeArray)
); /* Ожидаемый результат: массив отсортирован по возрастанию */
```
